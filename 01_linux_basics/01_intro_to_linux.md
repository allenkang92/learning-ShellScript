# 1.1 리눅스 소개

## 리눅스란?

리눅스는 자유-오픈 소스 운영 체제로, 1991년 리누스 토르발스가 개발을 시작했습니다. 유닉스를 모델로 하여 만들어졌으며, 다중 사용자, 다중 작업(멀티태스킹) 환경을 지원합니다.

## 리눅스의 주요 특징

1. **오픈 소스**: 소스 코드가 공개되어 있어 누구나 수정, 배포할 수 있습니다.
2. **안정성**: 서버 환경에서 높은 안정성을 제공합니다.
3. **보안성**: 다양한 보안 기능을 제공하며, 빠른 보안 패치가 이루어집니다.
4. **유연성**: 다양한 환경에 맞게 커스터마이징이 가능합니다.
5. **다양한 배포판**: Ubuntu, Fedora, CentOS 등 다양한 배포판이 존재합니다.

## 리눅스 배포판

리눅스 배포판은 리눅스 커널을 포함한 운영 체제 패키지를 의미합니다. 주요 배포판으로는:

- **Ubuntu**: 사용자 친화적인 인터페이스로 널리 사용됩니다.
- **Fedora**: 최신 기술을 빠르게 도입하는 배포판입니다.
- **CentOS**: 기업 환경에서 많이 사용되는 안정적인 배포판입니다.
- **Debian**: 안정성과 보안성이 뛰어난 배포판입니다.

## 리눅스를 배워야 하는 이유

1. **서버 관리**: 대부분의 웹 서버가 리눅스 기반으로 운영됩니다.
2. **개발 환경**: 많은 개발 도구와 언어가 리눅스에서 잘 동작합니다.
3. **클라우드 컴퓨팅**: AWS, Google Cloud 등의 클라우드 서비스가 리눅스 기반입니다.
4. **오픈 소스 생태계**: 다양한 오픈 소스 프로젝트에 참여할 수 있습니다.
5. **커스터마이징**: 시스템을 자신의 필요에 맞게 조정할 수 있습니다.
